# Azure Sentinel SIEM Lab

## Overview

This project demonstrates the deployment and configuration of Microsoft Sentinel and Azure Log Analytics for security monitoring and threat detection.

## Technologies

- Microsoft Sentinel
- Azure Monitor
- Log Analytics Workspace
- Kusto Query Language (KQL)

## Lab Activities

### Log Analytics Workspace
Created a Log Analytics Workspace named Sentinel-Lab.

### Microsoft Sentinel
Enabled Microsoft Sentinel on the workspace.
